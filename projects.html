These were projects I did for classes while attending the University of Texas at Austin.

CS378H: Distributed Computing and Systems Honors, Spring 2016

Snapshot Protocol Programming partner and I implemented a snapshot protocol based on "Efficient Distributed Snapshots" by Spezialetti and Kearns. We were able to create consistent global snapshots of an asynchronous system with any reason number of master nodes and a FIFO message delivery system. Our project was written in Java and made use of Java's networking and concurrency libraries.
